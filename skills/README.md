# Lab Skills

This folder holds the Claude Code skills the Innovation Lab builds and maintains. Skills are reusable knowledge bundles that orient Claude Code to lab conventions, methodologies, and workflows.

Each skill lives in its own subfolder with a SKILL.md describing what it does, when to use it, and how to invoke it.

## Status

The skills initiative is in early stages. As skills ship, they will be listed below with a brief description and link to the SKILL.md.

## Current skills

_None yet. As skills ship, they'll be listed here with a brief description and a link to their SKILL.md._

## Planned skills

- **friends-lab** — Encodes the orchestration pattern, domain-first build order, ADR triggers, and spinup conventions. The lab's signature skill.
- **federal-handoff** — Generates first drafts of submission package deliverables (technical document, README, pain point analysis, staffing approach) by reading the project's `/docs/` folder, codebase, and planning artifacts.
- **accessibility-audit** — WCAG and USWDS compliance checks.

This list will evolve as skills ship. Existing skills move out of "Planned" and into "Current skills" once their SKILL.md lands.
