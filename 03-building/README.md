# Building

Standards, tools, and starter prompts for building projects.

## Where to start on a new project

1. Read the project's `CLAUDE.md` — it has project-specific context
2. Run `npm run check` to verify everything passes before touching code
3. Pull the right starter prompt from [prompts/](prompts/) to get 50% there fast
4. Follow the [AI workflow](ai-workflow.md) for working with CC
