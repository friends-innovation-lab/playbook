# Building

Standards, tools, and starter prompts for building projects.

## Where to start on a new project

1. Read the project's `CLAUDE.md` — it has project-specific context
2. Run `npm run check` to verify everything passes before touching code
3. Reference [Storybook](storybook.md) when asking CC to build UI —
   paste the live Storybook URL alongside your Figma files for
   higher fidelity output
4. Need to update a color or spacing value? See
   [design-tokens.md](design-tokens.md)
5. Pull the right starter prompt from [prompts/](prompts/) to get 50% there fast
6. Follow the [AI workflow](ai-workflow.md) for working with CC
