# Tools and Access

Every tool the lab uses, what it does, and how to get access.

## Development

| Tool | What it does | Access |
|---|---|---|
| GitHub | Code hosting, CI/CD, project boards | Ask Lapedra — `friends-innovation-lab` org |
| Vercel | Frontend deployment, preview URLs | Ask Lapedra — `friends-innovation-lab` team |
| Supabase | Database and auth | Ask Lapedra — lab org |
| VS Code | Code editor | code.visualstudio.com |
| Node.js | JavaScript runtime | nodejs.org — use LTS |

## Monitoring and services

| Tool | What it does | Access |
|---|---|---|
| Sentry | Error tracking | sentry.io — `friends-innovation-lab` org |
| Resend | Transactional email | resend.com — lab account |
| Upstash | Rate limiting and caching | upstash.com — lab account |
| Vercel Analytics | Page views and performance | Included with Vercel |

## Communication and project management

| Tool | What it does | Access |

## Accounts that require Lapedra to set up

- GitHub org membership
- Vercel team membership
- Supabase org membership
- Figma org membership
- Sentry org membership
- Resend account access
- Upstash account access

**New to the lab?** Email or Slack Lapedra with your GitHub username and
the email you'll use for each service. She'll get you added.
