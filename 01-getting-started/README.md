# Getting Started

Everything you need to get set up and oriented before your first project.

## Steps

1. [First-time setup](first-time-setup.md) — Install tools and configure access. Do this once.
2. [What we build](what-we-build.md) — Understand the types of projects we work on.
3. [Tools and access](tools-and-access.md) — Every tool we use and how to get access.

Once you're set up, your first task on any project is to read the project's
`CLAUDE.md` file. That file tells you everything specific to that project.
